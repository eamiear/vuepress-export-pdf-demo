# 报表管理

## 集团经营报表

::: tip 描述
用于查询集团当日、单月、当年的经营数据状况
:::

### 今日数据

**操作步骤**

1. 进入系统: `报表管理`-`集团经营报表`，在右侧工具栏点击`今日`。
